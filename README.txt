Автономный сайт docs.techcatalyst.ru на Next.js.

Назначение:
- публичный раздел с информацией о программном продукте TechCatalyst AI;
- выдача подготовленных TXT и DOCX документов из соседней папки ../docs;
- самостоятельное развертывание через pm2.

Основные команды:
- npm install
- npm run build
- npm run start

PM2:
- pm2 start ecosystem.config.js

Порт по умолчанию:
- 3011
